Stops abuse of your site's Pingback method from XML-RPC by simply removing it.

== Description ==

Stops abuse of your site's Pingback method from XML-RPC by simply removing it. While you can use the rest of XML-RPC methods.

== Removed methods ==

* pingback.ping
* pingback.extensions.getPingbacks

= Requirements =

* WordPress 3.8.1 or higher.
    	
== Installation ==

* Extract the zip file and just drop the contents in the <code>wp-content/plugins/</code> directory of your WordPress installation (or install it directly from your dashboard) and then activate the plugin from Plugins page.
